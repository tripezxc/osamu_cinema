get_videos.php

Описание

Возвращает список видеозаписей

Параметры

Название Тип Описание
count integer. беззнаковое кол-во записей, которые будут возвращены. максимальное значение: 5
offset integer. беззнаковое смещение. номер записи с которой нужно начать

Возвращаемое значение

В случае успеха возвращает объект "videos", содержащий массив объектов типа video

Название Тип Описание
id int. беззнаковое. уникальный идентификатор фильма
title string. максимальная длина - 500 название фильма
description string. максимальная длина - 1000 краткое описание фильма
director string. максимальная длина - 500 режиссер фильма
year int. беззнаковое год выхода
logo string. максимальная длина - 1000 ссылка на постер фильма

В случае ошибки возвращает объект с полем error и описанием ошибки

Пример ответа при ошибке

{"error": "не удалось подключиться к базе данных"}
